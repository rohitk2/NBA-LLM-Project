# NBA Athlete Profile

This is a simple React app for managing NBA athlete profiles with scoring statistics and career descriptions.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open your browser and navigate to `http://localhost:3000`.

## Features

- Input field for NBA Athlete name.
- Editable career description.
- Scoring panel with fields for:
  - Points per game
  - Field goal percentage
  - Free throw percentage
